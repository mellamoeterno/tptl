---

In JavaScript, {} are optional when an if statement contains only
 one single statement.

So these two are equivalent:

if (size === "sm") sizeClass = "button-small"

and:

if (size === "sm") {
  sizeClass = "button-small"
}

challenge - button w/ variants

with logical conjunction we can use in the place of the ternary
operator we used in:

  let sizeClass = size ? `button-${size}` : ""

so with logical conjunction instead:

  let sizeClass = size && `button-${size}`


(&&) (logical conjunction) operator for a set of boolean 
operands will be true if and only if all the operands are true. 
Otherwise it will be false.

const a = 3;
const b = -2;

console.log(a > 0 && b > 0);
